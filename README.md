# 타입스크립트 학습 노트

## 1. 개요

### 요약
- 타입스크립트 작성은 일부 "추가 문서"와 함께 자바스크립트를 작성하는 것과 동일합니다.
- 타입스크립트는 브라우저 또는 노드에서 코드가 실행되는 방식에 영향을 미치지 않습니다.
- 타입스크립트는 코딩하는 동안 뒤에 앉아 있는 친구처럼 생각하는 것이 가장 좋습니다.
